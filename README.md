# Task Management API

This is an ongoing project to develop a Task Management API using GoLang.

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/task-management-api.git
   ```

2. **Install dependencies:**
   ```bash
   go mod download
   ```

3. **Run the project:**
   ```bash
   go run main.go
   ```
